HTML 5 Form Elements

  1. Required = Required input
  2. Placeholder = show the text when the input has no value
  3. Autofocus = automatically focus on input
  4. Value = by default value
  5. Read Only = you can real only this attribute, you cannot write anything in it but it will be submitted
  6. Disabled = not mutable, not focusable, or not even submitted with the form
  7. Checked = by default checked the input, you can unchecked it when you are filling the form
  8. Min = minimum numeric value
  9. Max = maximum numeric value
  10. Maxlength = used in textbox which shows maximum length of a text(input), it will not accept the text(characters) greater then the given length
